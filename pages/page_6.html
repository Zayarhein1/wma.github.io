HTML & CSS

HTML is the foundation behind all web pages. It’s used to add structure and form to text, images, and more. CSS is the language used to style HTML content. In this first course, you’ll learn the fundamentals of both languages so that you can create visually appealing web pages.

Style


There are three types of CSS. They are:

  • Inline
  • Internal
  • External


    • Inline

    Inline styles are plonked straight into the HTML tags using the style attribute. For Example, Using (style="color: red") to text. "Text" color wil be changed into the red.


    Internal

    Embedded, or internal, styles are used for the whole page. Inside the head element, the style tags surround all of the styles for the page.



    External

    External styles are used for the whole, multiple-page website. There is a separate CSS file, and is needed to be saved as “style.css” in the same directory as your HTML page then it can be linked to in the HTML like this:
